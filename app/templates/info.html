Sobre a Loja Virtual

Esta página apresenta um resumo do projeto, tecnologias utilizadas e instruções para rodar a aplicação localmente.

Funcionalidades

  • Catálogo de produtos com busca e paginação
  • Carrinho de compras
  • Checkout e pedidos
  • Avaliações de produtos
  • Autenticação de usuários

Como rodar

  1. Ative o ambiente virtual
  2. Instale as dependências: pip install -r requirements.txt
  3. Popule o banco: python populate_db.py
  4. Inicie o servidor: python run.py
